# How to run

## Setup

### Node

install node LTS from the node website

### pnpm

`npm i pnpm -g`

## Running

### Frontend

#### CD Frontend

`cd frontend`

#### Modules

`pnpm i`

#### Running

`pnpm dev`

### Backend

#### CD Backend

`cd backend`

#### Modules

`pnpm i`

#### running

`npm run dev`

### Done

#### Open the localhost on the port you got from the pnpm dev in frontend
